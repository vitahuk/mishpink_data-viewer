# Mishpink Data Explorer

A lightweight web application for processing and exploring user interaction data from the MishPink application.

Live: https://mishpinkprocessing.online
(password protected)

## Purpose
The application is designed exclusively for processing CSV data generated by the MishPink platform. It enables structured analysis of user sessions, interaction events, and derived metrics related to user behavior in map-based interfaces.

## Background
This application was developed in 2026 as part of a master's thesis focused on automated user testing of interactive geovisualizations.

The project was carried out in collaboration between the Department of Geography, Masaryk University and the Department of Geoinformatics, Palacký University Olomouc.

## Features
- Upload and processing of MishPink CSV data
- Session and group-level analysis  
- Computation of interaction metrics
- Basic browser-based data exploration

## Technology
- Python (FastAPI)  
- pandas  
- HTML / CSS / JavaScript  
- SQLite  

## ⚠️ Important Notice

This application operates on a shared live database.

- Any data modifications (including deletions) are permanent and immediately affect all users.
- The system uses SQLite, which supports only limited concurrent access (approximately up to 5 users at a time).
- Multiple users may interact with the same data simultaneously, which can result in changes being overwritten or removed during use.

Please use the application with caution, especially in multi-user scenarios.

## Notes
- This application is a research prototype and not intended as a production system.  
- Supported data format corresponds to MishPink exports only.
